# K-Slect 韓好物 — Design Context

## Design Context

### Users
台灣 20–35 歲女性，追求韓系生活風格。她們在手機上瀏覽、透過 LINE 社群發現新品，購買美妝、零食、服飾等日常韓貨。消費動機是「生活感升級」而非單純比價，重視視覺品味與信任感。

### Brand Personality
**清新、極簡、高級感**

介面應讓人感受到：「這家店有品味，我在這裡買東西不丟臉。」情感目標是自信（我做了好選擇）+ 輕鬆愉悅（瀏覽本身就是享受）。

### Aesthetic Direction
- **參考標竿**：Baiku（百貴）、MUSINSA — 乾淨的韓系排版、克制的留白、商品照片主導視覺
- **反例**：Shopee 橙橘紅、大字標討價促销感、塞滿的 banner 疊 banner
- **主題**：Light mode，以暖米白（#F7F6F3）為頁面底色，白色卡片，鼠尾草綠（#7C9070）為主要品牌色
- **字型**：Fraunces（標題 / 展示用，優雅帶有機感）+ Plus Jakarta Sans（內文 / UI，現代清晰）
- **色彩**：暖中性底色 + 鼠尾草綠作為可信賴的主色，赤陶色（#D4845E）僅用於折扣/促銷，避免濫用；不使用冷灰、不使用炫彩漸層

### Design Principles

1. **商品即主角** — 排版服務於商品圖片，UI 元素退到背景；大留白讓眼睛有地方停留
2. **克制的促銷感** — 折扣標籤存在但不喧嘩；用赤陶色低調標示，不用爆炸貼紙或大紅字
3. **行動優先，桌機也優雅** — 核心操作為單手拇指友善設計；桌機版本延伸為寬舒版面，而非手機版放大
4. **韓感不濫用** — 韓系美學透過字型選擇、顏色節制與構圖留白體現，而非貼韓文字或用 K-POP 視覺
5. **信任先於轉換** — 清晰的商品資訊、庫存狀態、配送說明；不用虛假倒數計時或過度催購
